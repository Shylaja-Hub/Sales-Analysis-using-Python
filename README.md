# 🛍️ Fashion Sales Data Analysis

## 📌 Project Overview

This project focuses on analyzing fashion retail sales data to extract meaningful insights and support data-driven decision making. The analysis includes data cleaning, exploratory data analysis (EDA), visualization, statistical testing, and dashboard creation.

---

## 🎯 Objectives

* Understand sales patterns and trends
* Identify key factors affecting sales
* Analyze the impact of discounts on sales
* Detect outliers and anomalies
* Apply statistical and probability concepts
* Build an interactive data analysis report

---

## 🛠️ Tools & Technologies

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Plotly
* Sweetviz

---

## 🔍 Project Workflow

### 1. Data Cleaning

* Handled missing values
* Converted data types
* Removed inconsistencies

### 2. Exploratory Data Analysis (EDA)

* Univariate and bivariate analysis
* Sales distribution analysis
* Category-wise performance

### 3. Data Visualization

* Used Matplotlib and Plotly
* Created interactive and static charts
* Identified trends and patterns

### 4. Outlier Detection

* Used box plots to identify extreme values
* Analyzed impact of outliers on sales

### 5. Hypothesis Testing

**Problem:** Does discount improve sales?

* Applied statistical testing
* Compared sales with and without discount
* Derived data-driven conclusions

### 6. Probability & Sampling Techniques

* Applied probability concepts to understand sales likelihood
* Used sampling methods to analyze subsets of data
* Explored different sampling techniques

### 7. Dashboard Creation

* Used Sweetviz to generate automated EDA reports
* Built a visual summary of the dataset

---

## 📊 Key Insights

* Discounts have a significant impact on increasing sales
* Certain categories perform better during specific periods
* Outliers influence overall sales trends
* Customer behavior varies based on pricing and discounts

---

## 🚀 Results

* Improved understanding of sales patterns
* Data-backed decision making for pricing and discounts
* Clear visualization of trends and anomalies
